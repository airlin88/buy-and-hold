## 🏗️ 第一部分：核心架構與環境架設 SOP (新手無痛指南)

本系統不依賴任何付費伺服器，而是利用目前全世界最穩定且免費的兩大科技巨頭服務：**GitHub** (作為前端網頁展示) 與 **Google 雲端硬碟** (作為資料庫與報價引擎)。

請按照以下 1、2、3 的順序，一步步建立您的雲端記帳本：

### 1. 前端介面：建立您的專屬網址 (GitHub Pages 流程)
這一步是為了讓您的手機或電腦隨時隨地都能連上這個精美的網頁介面。

1. **註冊 GitHub 帳號**：前往 [GitHub](https://github.com/) 免費註冊一個帳號。
2. **建立新的資料庫 (Repository)**：
   - 登入後，點擊右上角的 `+` 號 ➡️ 選擇 **`New repository`**。
   - `Repository name` 隨意填寫（例如：`buy-and-hold`）。
   - 請務必勾選 **Public** (公開)，這是使用免費網頁伺服器 (Pages) 的條件。
   - 點擊最下方的綠色按鈕 `Create repository`。
3. **上傳檔案**：
   - 在建立成功的頁面中，點擊 `uploading an existing file`。
   - 將我們提供給您的三個核心檔案：**`index.html`**、**`style.css`**、**`app.js`** 全部拖曳進畫面上方的虛線框內。
   - 點擊下方的綠色按鈕 `Commit changes`。
4. **開啟專屬網址**：
   - 點擊畫面上方的 `⚙️ Settings` 頁籤。
   - 在左側選單找到 `Pages`。
   - 在 `Build and deployment` 區塊，將 `Source` 的下拉選單改為 **`main`**，然後按 `Save`。
   - 等待約 2~3 分鐘後重新整理頁面，畫面上方就會出現您的專屬網址了！(例如：`https://您的帳號.github.io/buy-and-hold/`)

### 2. 後端大腦與雲端金庫：建立私有資料庫 (Google Apps Script)
這一步是為了建立一個安全存放交易紀錄的保險箱。這個保險箱就在您的 Google Drive 裡。

1. **建立空白試算表**：
   - 到您的 Google 雲端硬碟，新增一個空白的「Google 試算表」，命名為「存股族_雲端資料庫」。
   - 請查看這份試算表的網址列，複製 `/d/` 和 `/edit` 中間那一大串英數混合的亂碼。這串亂碼就是您的**試算表 ID**。
2. **打開 Google 程式碼引擎 (Apps Script)**：
   - 在試算表的頂部選單，點擊 `擴充功能` ➡️ `Apps Script`，會開啟一個寫程式的新分頁。
3. **貼上大腦程式碼**：
   - 把畫面中原本的 `function myFunction() { }` 全部刪除。
   - ✅ **複製並貼上以下整段程式碼：**

```javascript
/* ========================================================= 
   存股族 (Buy & Hold) - 資料庫後端大腦 (雲端金庫)
========================================================== */
const SPREADSHEET_ID = '★請貼上您的試算表ID★';
const SHEET_ASSETS = 'Assets';
const SHEET_TRANSACTIONS = 'Transactions';
const SECRET_PASSWORD = '★請設定您專屬的解鎖密碼★'; 

// 處理前端讀取資料 (GET 請求)
function doGet(e) {
  var pwd = e.parameter.pwd;
  if (pwd !== SECRET_PASSWORD) {
    return ContentService.createTextOutput(JSON.stringify({ "error": "Unauthorized" })).setMimeType(ContentService.MimeType.JSON);
  }

  var ss = SpreadsheetApp.openById(SPREADSHEET_ID);
  var result = { assets: [], transactions: [] };
  
  var sheetIdx = ss.getSheetByName(SHEET_ASSETS);
  if (sheetIdx) {
    var data = sheetIdx.getDataRange().getValues();
    if (data.length > 1) {
      for (var i = 1; i < data.length; i++) {
        result.assets.push({
          id: data[i][0] ? String(data[i][0]) : "",
          ticker: data[i][1] ? String(data[i][1]) : "",
          name: data[i][2] ? String(data[i][2]) : "",
          currency: data[i][3] ? String(data[i][3]) : "",
          maxInvested: Number(data[i][4]) || 0,
          totalCostBasisRealized: Number(data[i][5]) || 0,
          realizedPnL: Number(data[i][6]) || 0,
          firstDate: data[i][7] ? data[i][7] : null,
          lastDate: data[i][8] ? data[i][8] : null
        });
      }
    }
  }
  
  var sheetTxs = ss.getSheetByName(SHEET_TRANSACTIONS);
  if (sheetTxs) {
    var tData = sheetTxs.getDataRange().getValues();
    if (tData.length > 1) {
      for (var i = 1; i < tData.length; i++) {
        result.transactions.push({
          id: tData[i][0] ? String(tData[i][0]) : "",
          assetId: tData[i][1] ? String(tData[i][1]) : "",
          type: tData[i][2] ? String(tData[i][2]) : "",
          date: tData[i][3] ? tData[i][3] : null,
          price: Number(tData[i][4]) || 0,
          quantity: Number(tData[i][5]) || 0,
          fee: Number(tData[i][6]) || 0,
          currency: tData[i][7] ? String(tData[i][7]) : "",
          originalPrice: tData[i][8] ? Number(tData[i][8]) : null,
          exchangeRate: tData[i][9] ? Number(tData[i][9]) : null
        });
      }
    }
  }
  return ContentService.createTextOutput(JSON.stringify(result)).setMimeType(ContentService.MimeType.JSON);
}

// 處理前端寫入資料 (POST 請求)
function doPost(e) {
  var payload;
  try { payload = JSON.parse(e.postData.contents); } catch(err) { return ContentService.createTextOutput(JSON.stringify({"error": "Invalid JSON"})).setMimeType(ContentService.MimeType.JSON); }

  var pwd = payload.pwd;
  if (pwd !== SECRET_PASSWORD) { return ContentService.createTextOutput(JSON.stringify({ "error": "Unauthorized" })).setMimeType(ContentService.MimeType.JSON); }

  var ss = SpreadsheetApp.openById(SPREADSHEET_ID);
  
  var s1 = ss.getSheetByName(SHEET_ASSETS);
  if (!s1) s1 = ss.insertSheet(SHEET_ASSETS);
  s1.clear();
  var assetRows = [["id", "ticker", "name", "currency", "maxInvested", "totalCostBasisRealized", "realizedPnL", "firstDate", "lastDate"]];
  if (payload.assets) { payload.assets.forEach(function(a) { assetRows.push([a.id||"", a.ticker||"", a.name||"", a.currency||"", a.maxInvested||0, a.totalCostBasisRealized||0, a.realizedPnL||0, a.firstDate ? new Date(a.firstDate) : "", a.lastDate ? new Date(a.lastDate) : ""]); }); }
  if (assetRows.length > 0) s1.getRange(1, 1, assetRows.length, assetRows[0].length).setValues(assetRows);
  
  var s2 = ss.getSheetByName(SHEET_TRANSACTIONS);
  if (!s2) s2 = ss.insertSheet(SHEET_TRANSACTIONS);
  s2.clear();
  var txRows = [["id", "assetId", "type", "date", "price", "quantity", "fee", "currency", "originalPrice", "exchangeRate"]];
  if (payload.transactions) { payload.transactions.forEach(function(t) { txRows.push([t.id||"", t.assetId||"", t.type||"", t.date ? new Date(t.date) : "", t.price||0, t.quantity||0, t.fee||0, t.currency||"", t.originalPrice||"", t.exchangeRate||""]); }); }
  if (txRows.length > 0) s2.getRange(1, 1, txRows.length, txRows[0].length).setValues(txRows);
  
  return ContentService.createTextOutput(JSON.stringify({ "status": "success" })).setMimeType(ContentService.MimeType.JSON);
}

function doOptions(e) { return ContentService.createTextOutput("OK").setMimeType(ContentService.MimeType.TEXT); }
```

4. **設定密碼與 ID**：

   - 將第 7 行的 `★請設定您專屬的解鎖密碼★` 替換為您自訂的密碼（例如 `mypassword123`）。

### 3. 即時報價引擎：架設 GoogleFinance 代理站 (第二支 GAS)
這一步非常關鍵。由於瀏覽器通常不允許跨網域直接查詢股價，我們利用 Google 自家強大的 `=GOOGLEFINANCE` 函式，打造一個專屬的報價中繼站。

1. **建立新的 Apps Script 專案**：
   - 請直接前往 [Google Apps Script 首頁](https://script.google.com/home)。
   - 點擊左上角的 **`新專案`** 按鈕。
   - 將專案名稱改為「即時報價引擎」。
2. **貼上報價引擎程式碼**：
   - ✅ **把預設程式碼刪除，複製並貼上以下整段避開快取與限制的 Google Finance 報價轉換器：**

```javascript
/* ========================================================= 
   存股族 (Buy & Hold) - 即時報價代理伺服器 (=GOOGLEFINANCE)
========================================================== */

function doGet(e) {
  // 支援 ?q=QQQ,VOO 參數呼叫
  var q = e.parameter.q || (e.parameters && e.parameters.q ? e.parameters.q[0] : null);
  if (q) {
    return fetchGoogleFinanceAPI(q);
  }
  return ContentService.createTextOutput("Invalid Request").setMimeType(ContentService.MimeType.TEXT);
}

function fetchGoogleFinanceAPI(tickersStr) {
  var tickers = tickersStr.split(',');
  var result = {};
  
  var ss = SpreadsheetApp.create("Temp_Quotes_" + new Date().getTime());
  var sheet = ss.getSheets()[0];
  
  for (var i = 0; i < tickers.length; i++) {
    var symbol = tickers[i].trim();
    if (!symbol) continue;
    
    // 台股代碼轉換 (例如：0050 -> TPE:0050)
    var querySymbol = symbol;
    if (/^\d{4}$/.test(symbol) || /^\d{4}[A-Za-z]$/.test(symbol)) {
      querySymbol = "TPE:" + symbol;
    }
    
    // 匯率代碼轉換 (為相容前端，將 USDTWD=X 轉給 Google 看得懂的格式)
    if (symbol === 'USDTWD=X') {
      querySymbol = "CURRENCY:USDTWD";
    }
    
    var formula = '=IFERROR(GOOGLEFINANCE("' + querySymbol + '", "price"), "")';
    sheet.getRange(i+1, 1).setValue(symbol);
    sheet.getRange(i+1, 2).setFormula(formula);
  }
  
  try {
    SpreadsheetApp.flush(); // 強制 Google 計算公式
    Utilities.sleep(1500);  // 給予 Google API 計算時間
    
    var data = sheet.getDataRange().getValues();
    for (var j = 0; j < data.length; j++) {
      var t = data[j][0];
      var p = data[j][1];
      if (p !== "" && typeof p === 'number') {
        result[t] = parseFloat(p);
      }
    }
  } catch(err) {
    // 發生超時或計算失敗
  } finally {
    DriveApp.getFileById(ss.getId()).setTrashed(true); // 用完就立刻銷毀暫存檔，不留痕跡
  }
  
  // 建立 CORS 標頭讓前端可讀取
  return ContentService.createTextOutput(JSON.stringify(result))
         .setMimeType(ContentService.MimeType.JSON);
}

function doOptions(e) {
  return ContentService.createTextOutput("OK").setMimeType(ContentService.MimeType.TEXT);
}
```

   - 選 **`網頁應用程式`**，將誰可以存取設定為 **「所有人 (Anyone)」** ➡️ 按下部署並授權。

## 📱 二、系統日常使用方式

- **安全登入與斷線**：打開您剛架好的專屬網址時，會看到科技感防彈玻璃登入艙。輸入密碼後即可解鎖。系統會自動暫存安全憑證在您的裝置上，往後可以直接進入。
- **新增交易**：請點擊底部導覽列切換至 **「歷史」** 頁籤，接著點擊畫面右上角的 **`+ 新增紀錄`** 按鈕。系統支援**台股 (TWD)** 與 **美股 (USD)** 交易。
- **重新整理與即時報價**：點擊右上角的 `🔄 重新整理按鈕`，系統會向您的即時報價引擎 (Google Apps Script) 發送請求，利用 `=GOOGLEFINANCE` 的強大功能即時帶回最新的股價與匯率，接著網頁會自動為您結算總資產與未實現損益。
- **資產配置 (V1.1 新增)**：在總覽頁面下方的「📊 資產配置」卡片中，可查看您的全方位資產分布。點擊各類別（現金/股票/期貨/債券）可展開明細。現金、期貨、債券可自訂帳戶名稱與金額；股票資料自動從庫存系統帶入，並細分為「台股 ETF」、「個股」與「美股 ETF」三個子分類。
- **Excel 資料備份**：點擊右上角的 `📁 資料庫按鈕`。您可以匯出所有交易紀錄回私人電腦備份；如果未來換手機不想重新連線抓取，也可以在此上傳 Excel（注意，這會徹底覆蓋現有資料）。

## 💻 三、未來系統升級與除錯指南

### 1. 遇到手機版介面卡死，或是改了程式沒反應？
如果您修改了 `app.js` 或 `style.css` 並重新上傳 GitHub，但手機瀏覽器依然死讀舊畫面：
1. 請打開您的 `index.html`。
2. 找到 `<link rel="stylesheet" href="style.css?v=2">` 與 `<script src="app.js?v=2"></script>`。
3. 把最後的參數（例如 `?v=2`）改成新的版本號碼（例如 `?v=3`），存檔重新上傳，這能直接突破地表上所有瀏覽器（特別是 Apple iOS Safari）的頑固快取 (Cache Busting)！

### 2. 想換密碼該怎麼做？
您無須更動任何前端網頁程式碼：
1. 回到 Google Drive，打開「存股族_雲端資料庫」試算表裡的 `擴充功能 > Apps Script`。
2. 在第 10 行直接更改密碼文字。
3. **⚠️ 絕對不可忘記的步驟**：點擊右上角 `部署` ➡️ `管理部署作業` ➡️ 點擊左側部署版本，再點擊右側的`✏️ 鉛筆圖示 (編輯)` ➡️ 將「版本」選單改為 **「建立新版本 (New Version)」** ➡️ 按下部署。
*(如果少了「建立新版本」這個動作，您的密碼修改永遠都不會生效。)*

### 3. 可以手動竄改後台試算表的數字嗎？
沒問題！這套系統的最高管理員就是您：
1. 直接進入 Google Drive 打開試算表。
2. 切換到 `Transactions` 工作表，直接更改、刪除裡面的數字，就像一般 Excel 一樣操作。
3. 只要不要動到第一排的英文欄位名稱 (Header)，當使用者重整網頁時，看見的就是您干預後的正確數字。這是遇到任何 Bug 或記錯帳時，最暴力的終極修復方式。
